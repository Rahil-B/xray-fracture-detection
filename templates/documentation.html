Documentation

Methodology

Our project utilized a combination of machine learning and deep learning techniques to develop an efficient wrist anomaly detection system for pediatric physicians and surgeons. We performed a total of 23 detection procedures, which involved data preprocessing, image augmentation, and experimentation with different architectures.

Neural Network Models

We explored state-of-the-art YOLO (You Only Look Once) models, including YOLOv5, YOLOv6, YOLOv7, and YOLOv8 for wrist anomaly detection. Among these models, YOLOv8x was identified as the best-performing variant, especially for detecting small abnormalities in pediatric X-ray images.

To utilize YOLO models efficiently, we integrated the Ultralytics PyTorch library (GitHub Repository) into our project. This library simplified model implementation, training, and evaluation, allowing us to focus on enhancing detection accuracy and optimizing the system's performance.

Faster R-CNN for Comparison

In addition to YOLO models, we also experimented with Faster R-CNN (Region-Based Convolutional Neural Network), a two-stage object detection model. While YOLO excels in real-time detection, Faster R-CNN provides a strong baseline for comparison in our research.

Flask Web Interface

To facilitate easy access to the wrist anomaly detection system, we developed a user-friendly web-based interface using Flask. The interface allows medical professionals to upload pediatric X-ray images, and the AI system processes the images and highlights any detected wrist anomalies with bounding boxes.

Additionally, we implemented several functions within the Flask application, such as:

  • Function 1: Image Preprocessing - prepares the uploaded X-ray images for input to the YOLO model.
  • Function 2: Anomaly Detection - performs object detection using the trained YOLOv8x model and visualizes anomalies with bounding boxes.
  • Function 3: Anomaly Counting - counts the total number of wrist anomalies detected in the X-ray image.
  • Function 4: Transfer Learning - fine-tunes the YOLO model for extending anomaly detection to other bones.

Android Application

To ensure broader accessibility, we developed an Android application that integrates the wrist anomaly detection system. The Android app allows medical professionals to perform wrist anomaly diagnosis using their mobile devices. It features a user-friendly interface with functionalities similar to the web-based version.
